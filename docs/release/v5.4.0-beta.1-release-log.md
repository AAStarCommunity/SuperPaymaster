- [x] **RPC ghost-nonce incident & recovery (the lesson, in practice):**
  - **Alchemy accepted** the deploy txns via `eth_sendRawTransaction` but **never broadcast** them
    → local nonce advanced, network nonce did not → **`forge --slow` hung** waiting for receipts.
  - **Diagnosed** by `curl`-ing `eth_getTransactionCount(latest)` on **publicnode** — network nonce was
    **clean** (no pending), confirming the txns were ghosts stuck at Alchemy.
  - **Rebroadcast via publicnode succeeded**; all txns then mined.
- [x] **Nonce delta +9, all mined** (see Phase 4).
