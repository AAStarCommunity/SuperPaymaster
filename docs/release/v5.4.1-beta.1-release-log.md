## Phase 9 — Post-release

- [ ] Open `bugfix/post-v5.4.1` tracking branch
- [x] **Lessons learned** from this release:
  1. **Ghost nonce (Alchemy)**: Alchemy Sepolia accepts TX without broadcasting → duplicate nonces. Mitigation: use `--private-key` with `--slow` for UUPS upgrade CALLs, or switch to publicnode for broadcasts.
  2. **Version bump triggers UUPS upgrade**: changing `version()` string changes bytecode hash → deploy-core correctly detects and runs UpgradeLive. Nonce collision prevented auto-upgrade; fixed with manual `cast send upgradeToAndCall`.
  3. **forge fmt MockUSDT**: pre-existing import format diff appears on `forge fmt --check` but is from library, not our code — can be resolved by running `forge fmt` once.
  4. **Registry version didn't bump**: Registry bytecode didn't change (no storage/logic change), so its `version()` stays `Registry-5.4.0`. Only SuperPaymaster impl changed.
