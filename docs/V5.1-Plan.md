## 1. Objective

Extract the universal billing kernel `_consumeCredit()` from `postOp`, add `chargeMicroPayment()` as a new EIP-712 signed payment entry, and optimize batch gas with EIP-1153 transient storage.

**Key Constraint**: `postOp` external behavior must be **identical** to V4.1.0. Zero regression.

**Key Insight (Design Doc §7.5)**: V5.1 **80%+ 基于已有模块复用**，新增代码量仅 ~125 行合约 + ~300 行测试：

| V5.1 新能力 | 复用的已有模块 | 新增代码 |

| Debt recording | `xPNTsToken.recordDebt()` (**完全复用**) | 0 行 |

### T3: Implement `chargeMicroPayment()` (Week 2)

**Description**: New external function for non-ERC-4337 micropayment.

**Interface**:

```solidity

    address operator,
    address user,
    uint256 usdAmount,
    uint256 nonce,
    uint256 deadline,
    bytes calldata signature
) external nonReentrant;
```

**Sub-tasks**:

| # | Task | Est. | File |
|---|------|------|------|
| T3.1 | Implement signature verification: **`SignatureCheckerLib.isValidSignatureNow()`** (支持 EOA + ERC-1271 智能账户，如 AirAccount) | 2h | SuperPaymaster.sol |
| T3.2 | Implement nonce management (sequential increment + replay protection) | 1h | SuperPaymaster.sol |
| T3.3 | Implement deadline check | 0.5h | SuperPaymaster.sol |
| T3.4 | **新增 `_calculateAPNTsAmountFromUSD(uint256 usdAmount)` 内部函数**（USD→aPNTs 直接转换，使用 `aPNTsPriceUSD`，区别于 gas→aPNTs 的 `_calculateAPNTsAmount`） | 1.5h | SuperPaymaster.sol |
| T3.5 | Implement identity check: `sbtHolders[user]`（V5.3 扩展为 `isEligibleForSponsorship()`） | 0.5h | SuperPaymaster.sol |
| T3.6 | Implement operator availability check (`isConfigured` + `!isPaused`) | 0.5h | SuperPaymaster.sol |
| T3.7 | **复用 Rate Limiting**: `userOpState[operator][user].lastTimestamp` + `minTxInterval` 检查 (Design Doc §7.4 step 7) | 1h | SuperPaymaster.sol |
| T3.8 | Call `_consumeCredit(operator, user, aPNTsAmount, false)` | 0.5h | SuperPaymaster.sol |
| T3.9 | Add `MicroPaymentCharged` event | 0.5h | SuperPaymaster.sol + ISuperPaymaster.sol |
| T3.10 | Update `ISuperPaymaster.sol` interface | 0.5h | ISuperPaymaster.sol |

**Logic Flow** (Design Doc §7.4 完整 8 步):

```
1. if (block.timestamp > deadline) revert ExpiredSignature()
2. if (microPaymentNonces[user] != nonce) revert InvalidNonce()
3. microPaymentNonces[user]++
4. Verify EIP-712: digest = _hashTypedData(structHash)
   → SignatureCheckerLib.isValidSignatureNow(user, digest, signature)
   → 支持 EOA (ecrecover) AND 智能账户 (ERC-1271, 如 AirAccount)
5. Identity check: if (!sbtHolders[user]) revert UnauthorizedUser()
6. Operator check: if (!config.isConfigured || config.isPaused) revert OperatorNotAvailable()
7. Rate limiting: if (block.timestamp - state.lastTimestamp < minTxInterval) revert RateLimited()
   → state.lastTimestamp = uint48(block.timestamp)
8. aPNTsAmount = _calculateAPNTsAmountFromUSD(usdAmount)
   → _consumeCredit(operator, user, aPNTsAmount, false)
   → emit MicroPaymentCharged(operator, user, usdAmount, aPNTsAmount, nonce)
```

**`_calculateAPNTsAmountFromUSD()` 说明**:
```solidity
/// @dev Convert USD amount to aPNTs using aPNTsPriceUSD
/// Different from _calculateAPNTsAmount() which converts Gas cost
function _calculateAPNTsAmountFromUSD(uint256 usdAmount) internal view returns (uint256) {
    // usdAmount is in 18 decimals, aPNTsPriceUSD is in 18 decimals
    // e.g., $0.10 = 0.1e18, aPNTsPriceUSD = 0.02e18
    // result = 0.1e18 * 1e18 / 0.02e18 = 5e18 (5 aPNTs)
    return (usdAmount * 1e18) / aPNTsPriceUSD;
}
```

**Acceptance Criteria**:

- [ ] Valid EIP-712 signature from EOA user → successful charge
- [ ] Valid EIP-712 signature from **smart account** (ERC-1271, e.g. AirAccount) → successful charge
- [ ] Invalid signature → revert `InvalidSignature`
- [ ] Expired deadline → revert `ExpiredSignature`
- [ ] Wrong nonce → revert `InvalidNonce`
- [ ] Replay same nonce → revert
- [ ] Operator insufficient balance → revert `InsufficientOperatorBalance`
- [ ] Operator paused → revert `OperatorNotAvailable`
- [ ] Operator not configured → revert `OperatorNotAvailable`
- [ ] User without SBT → revert `UnauthorizedUser`
- [ ] **Rate limited**: same user+operator within `minTxInterval` → revert `RateLimited`
- [ ] USD→aPNTs conversion correct: $0.10 at aPNTsPriceUSD=0.02 → 5 aPNTs
- [ ] Correct `protocolRevenue` increment
- [ ] Correct `xPNTsToken.recordDebt()` call
- [ ] `MicroPaymentCharged` event emitted with correct params

