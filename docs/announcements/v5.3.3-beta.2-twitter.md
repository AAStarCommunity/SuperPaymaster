🚀 SuperPaymaster v5.3.3-beta.2 — security-hardened beta live on Sepolia.

A thread on what's live 🧵

**2/ Gasless sponsorship**
```
⛽ Communities sponsor members' gas with their own token (xPNTs) — users hold zero ETH.

Two modes:
• AOA+ — one shared paymaster, many operators
• AOA — independent per-community paymaster (EIP-1167 clones)
```

**3/ Human + Agent**
```
🤖 Dual-channel eligibility: sponsor a human (SBT holder) OR an AI agent (ERC-8004 registered).

Same paymaster, same credit rails — one for people, one for autonomous agents.
```

**4/ Reputation → Credit**
```
💳 Sponsorship isn't unlimited. A user's credit/debt ceiling is driven by reputation:

• Global Reputation (Registry) → credit limit
• Community Reputation (per-community rules)

Pay from xPNTs balance, or draw down credit up to your limit.
```

**5/ Streaming + x402**
```
🌊 MicroPaymentChannel — open a channel, sign vouchers off-chain, settle in batch. The session/streaming model for agent micropayments.

🔗 x402 — per-request settlement (EIP-3009 USDC + xPNTs direct). Contracts live; ~~SDK signing in progress (#39)~~ SDK available @aastar/sdk@0.29.0.
```
